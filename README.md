# OS 
This is a project that aims to develop a unix-like x86_64 operating system that supports windows, linux & macOS executable. 
## What has been done
Right now, the project is still in it's very early stages, and I'm focusing on building a reliable bootloader before actually starting to write the real operating system kernel. I've implemented the switch from real to long mode, PCI & AHCI drivers + a couple of generic functions inside the bootloader to print stuff on the screen for example.

99.999% of the work.
